# Weight-converter
Weight Converter
A lightweight and user-friendly Python application that converts weights between Kilograms (Kg) and Pounds (Lbs).
This project demonstrates both command-line (CLI) and graphical user interface (GUI) implementations using Tkinter, showing a smooth progression from a basic Python script to a professional desktop app.
